# Getting Started with Jx

Jx lets you build entire websites using JSON. No complex build tools, no framework lock-in — just declarative documents that compile to static HTML.

## Why Jx?

- **Declarative** — describe what you want, not how to build it
- **Zero runtime** — static HTML output by default
- **Content collections** — Markdown, JSON, and CSV content with schema validation
- **Layouts and slots** — reuse page shells across your site

Start by creating a `project.json` in your project root.
