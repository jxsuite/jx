## How It Works

1. **Write JSON** — Define components, pages, and behavior as structured JSON documents. No JSX, no template language, no build-step syntax to learn.

2. **Compile** — The JX compiler produces static HTML with zero JavaScript by default. Reactive islands hydrate only where you need interactivity.

3. **Deploy** — Ship to any static host — Netlify, Vercel, Cloudflare Pages, or a plain web server. No server runtime required.
